While there are many tools for capturing screenshots of web pages, the image created with https://snapito.com has a URL link to the screenshot that includes the original web address in it. I'm sure there are better solutions to the one I'm going to describe that the tech experts will know and hopefully provide, but in a pinch (assuming you are using a Windows based PC) you can always hit Control (Cntrl) and Print Screen (PrtSc) buttons, then open Paint program, and hit Cntrl V (or paste from the pull down menu), and the screen shot will appear in Paint, and you can then save it as a Paint document. Then you could upload those Paint images to imgur or some other photo hosting platform.

Hitting Cntrl / PrntSc copies exactly whatever is on your screen at that moment.

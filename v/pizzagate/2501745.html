Validation vs. soundness An argument is sound if and only if

The argument is valid, and All of its premises are true. For instance,

All men are mortal. Socrates is a man. Therefore, Socrates is mortal. The argument is valid (because the conclusion is true based on the premises, that is, that the conclusion follows the premises) and since the premises are in fact true, the argument is sound.

The following argument is valid but not sound:

All organisms with wings can fly. Penguins have wings. Therefore, penguins can fly.

Since the first premise is actually false, the argument, though valid (the premises of an argument do not have to be true in order for the argument to be valid), is not sound.

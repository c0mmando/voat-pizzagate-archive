argosciv 2020-06-23T00:57:10

Short answer:

Convenience for accessing and distributing all files regardless of which page of the website displays the files' content.

Longer answer:

The files are stored in a particular directory which is accessible via file.wikileaks.org. Visiting that link directly lists the files index of the folder which that subdomain points to. In that folder we have a folder called file. In that folder, we find the list of folders which contains files corresponding to particular publications/dumps/categories.

If Wikileaks wanted to disable the ability to browse this file/folder index, it would be incredibly easy to disable the default index display functionality (a very rudimentary process in terms of website development). They have chosen not to, presumably for the aforementioned purpose of user convenience.

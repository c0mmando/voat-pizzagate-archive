throwawaa 2017-01-17T03:13:17

EDIT: the curl command in this script doesn't seem to download the right thing. Still figuring it out.

Here's a simple bash script for downloading zips of all the archive.is files in a given text document. I'm using this to keep a local copy of all the archive.is zips in my local research doc, but one could easily make a script that will scan the voat threads and download the zips in them as well. 

function download_archive_zips() {
echo "downloading archive.is zips found in $1"
string=`cat $1`
pattern="http://archive.is/[a-zA-Z]{5}
for word in $string
do
    [[ $word =~ $pattern ]]
    if [[ ${BASH_REMATCH[0]} ]]
    then
        match="${match:+match }${BASH_REMATCH[0]}"
       symbol=`echo "$match" | sed 's/.*\///'`
       sleep 1
       curl -o ~/$symbol.zip "http://archive.is/download/${symbol}.zip"
    fi
done
}

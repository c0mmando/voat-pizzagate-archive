Perhaps groups of you are organizing elsewhere, but I wanted to suggest this online tool for organizing research. Trello.com You could set up Boards - a personal board for your own research, a Lurker Board to make public with memes and infographics and a Researcher Board for collaboration. You can set up teams and invite various teams to work on each board. Each board can contain a List with Cards. that can be moved from list to list... Each Card can contain Tasks can be assigned, comments...Lots of different ways to organize. Just thought it might help some of you...

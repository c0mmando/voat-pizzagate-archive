As it appears the coverup is really going after the reasearch now, here are some tools to fight back. ( pizzagate )

submitted 2017-01-16T19:50:57 by ThePizzaInvestigator

I.P.F.S. - This is a distributed internet that will be highly resistant to takedowns and other attacks, good for archiving stuff.

T.O.R. - When used properly, allows full anonymity from even the government, while using this, make sure it's set to the highest security and that javascript is always off. Do not log into any accounts associated with you during the same session that you're accessing anonymous accounts, always generate a new identity between accessing things you want to remain unlinked. I do not recommend using TOR on Windows 8 or above.

T.A.I.L.S. - An OS Designed specifically around TOR and privacy, If you have windows 8, 10, put this on a memory stick and use it when you want to be secure. While you should still have javascript disabled, it is much more secure under TAILS than it is in Windows. Do not use in a virtual machine under windows, under Linux is more or less OK though.

Tox - A fully end to end encrypted messaging system, works best when paired with TOR

Linux - Kind of obvious, but, if you didn't know already, windows is full of backdoors and other risks, stay away from Ubuntu, as they've been untrustworthy in the past. I recommend using Manjaro

As always, stay safe, be secure, don't let the government get you.

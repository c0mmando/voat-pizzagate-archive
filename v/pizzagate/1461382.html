NoRagrets 2016-12-05T01:24:30

Create a few bookmarks and change the URL (destination/web page) to the following javascripts.

While viewing an image, this one will import the image into https://sli.mg
javascript:void(open('https://sli.mg/upload?url='+encodeURIComponent(document.location)))

This one is similar to above but skips the confirmation
javascript:void(open('https://sli.mg/upload?url='+encodeURIComponent(document.location)+'&auto=true'))

This script will automatically archive whatever webpage you are currently viewing with https://archive.is
javascript:void(open('https://archive.is/?run=1&url='+encodeURIComponent(document.location)))

Edit: Here is an example of my auto-confirm, sli.mg import bookmark in Firefox.
https://i.sli.mg/g8MvCO.png

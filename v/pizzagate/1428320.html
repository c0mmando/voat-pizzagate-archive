submitted 2016-11-23T01:01:14 by Drain0

So when I finally left Reddit for good somebody recommended that I find a script on-line to clean up behind myself. I found a script that goes through your comment history and overwrites everything. It suggests that you use RES and scroll to the bottom to open up all pages of comments.

I installed it on Opera using Tampermonkey and it's working perfectly. I did not write the script so I am not taking credit for it. I just wanted to let people be aware that the tools are out there. There are others out there along with instructions on how to install them and how to use them, but this one is working for me so I haven't tested any others.

Reddit Comment Overwrite Script

If you need instructions on how to install userscripts on your browser, go HERE

It is highly recommended to use RES with it's infinite scroll capability turned on to reach the bottom of your comments so you don't have to manually go page by page. Reddit Enhancement Suite

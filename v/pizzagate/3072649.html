How to Archive Facebook Comments ( pizzagate )

submitted 2019-03-03T20:23:31 by IShallNotFear

This is a companion post to my recent write up about James Alefantis' Paris trip in 2015. https://voat.co/v/pizzagate/3071535

When I was working on my last post, I couldn't get the Facebook comments to archive. I tried all of my normal methods (screen shot, cache, translate, archiving on one site and then archiving that on another site) but nothing was showing up correctly. For example, I wanted to archive James Alefantis' response on this post. https://www.facebook.com/artem.shcherbakov.96/posts/10155422228960570 However, you can't see the comments on archive.is ( http://archive.is/kducw ) or on web.archive.org ( http://web.archive.org/web/20190122031207/https://www.facebook.com/artem.shcherbakov.96/posts/10155422228960570 ). This is because the comments require an additional click to open on a computer. However, this is not the case on a phone. When I switched the links to the mobile version, the comments appeared. All I had to do was replace "https://www" with "m" ( https://m.facebook.com/artem.shcherbakov.96/posts/10155422228960570 ). This archived perfectly. ( http://archive.li/kIBn6 ) I was then able to copy the link from "view previous comments" in order to archive comments that didn't fit on the first page. ( http://archive.li/ybT7P )

This one isn't a complex fix. However, I just thought I would make a post for those who come across this issue later. No need to struggle as much as I did.
